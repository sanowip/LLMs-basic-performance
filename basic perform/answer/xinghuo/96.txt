In supervised learning, which is a type of machine learning that uses labeled data, the model is trained on a dataset where each input (a sequence of words) is paired with the correct output (the next word in the sequence). The model learns to predict the correct output based on the patterns it sees in the input data.

In unsupervised learning, which is a type of machine learning that uses unlabeled data, the model is trained on a dataset without any labels. Instead, the model learns to identify patterns and relationships in the data on its own. This can be done by training the model on a large corpus of text and allowing it to learn from the statistical properties of the language.
